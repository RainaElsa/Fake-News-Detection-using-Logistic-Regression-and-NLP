# Fake News Detection using Logistic Regression

This project utilises machine learning and NLP techniques to classify news articles as either **Fake** or **Real**.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLP, TF-IDF

## Approach
- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression model
- Accuracy-based evaluation

## Dataset
Publicly available Fake News dataset(https://www.kaggle.com/datasets/rajatkumar30/fake-news)
